### v1 데모 세션 피드백 (v2 추가)
- **형식**: 직접 코드 변경 기록 (13건, CC-001~CC-013)
- **분석 일시**: 2026-04-01
- **추출된 요구사항**:
  - 지도 범례 추가 (중요도별 색상 + 항로 선 설명)
  - NOTAM 원형 클릭 시 팝업 (상세 정보 + 상세 페이지 링크)
  - 항로 드롭다운에 "전체" 옵션 추가
  - "중요 NOTAM" 범위를 critical + high로 확장
  - 지도에 해당 항로의 모든 NOTAM 표시 (critical만이 아닌 전체)
  - 상태 컬럼에 info 팝오버로 상태값 설명 제공
  - NOTAM 중요도별 일관된 색상 코딩 (빨강/주황/파랑)
  - 지도 헤더에 설명 팝오버 추가
- **추출된 페인 포인트**:
  - 지도 마커/원형의 의미가 불명확하여 사용자 혼란
  - NOTAM 상세 정보 확인을 위해 별도 네비게이션 필요
  - 상태값의 의미를 사용자가 직관적으로 파악하기 어려움
  - Critical NOTAM만 표시하면 High 중요도 정보가 누락되어 의사결정에 불완전
- **추출된 컨텍스트**:
  - Leaflet 지도 라이브러리의 webpack 호환성 이슈 존재
  - Cloudscape Popover 컴포넌트가 info 팝오버 패턴에 활용됨
- **신뢰도**: 높음 (실제 데모 중 사용자 피드백에 기반, 이미 코드에 반영 완료)
